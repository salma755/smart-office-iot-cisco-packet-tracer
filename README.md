# Smart Office IoT System

A smart office automation system designed and simulated using Cisco Packet Tracer as part of a university networking and IoT course.

The project demonstrates basic IoT concepts and device connectivity within a smart environment.

## Project Overview

The system simulates a smart office environment using IoT devices, where multiple components are connected to enable basic automation and monitoring.

The project includes:

- Smart lighting system
- Basic IoT device connectivity
- Network configuration using Cisco Packet Tracer
- Simple automation logic within the simulation environment

## Technologies

- Cisco Packet Tracer
- IoT Simulation Concepts
- Basic Networking (IP configuration, device connectivity)

## Project Context

This project was developed as a final course assignment to explore the fundamentals of IoT systems and networked smart environments.

While the implementation is at an introductory level, it provided valuable hands-on experience in designing and simulating connected systems.

## Challenges

The main challenge was understanding and properly configuring IoT device interactions within Cisco Packet Tracer, as well as achieving correct network connectivity between components.

Despite limitations, the project successfully demonstrated a functional smart office concept.

## Learning Outcomes
- Introduction to IoT systems
- Network configuration basics
- Simulation using Cisco Packet Tracer
- Understanding smart environment concepts
